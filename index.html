Privacera Base Chart

A comprehensive base Helm chart for microservices with built-in observability, security, and scalability features

v1.0.0 Kubernetes Microservices Observability

📦 Installation

Add this Helm repository and install the chart:

helm repo add privacera-base https://evilgenius-jp.github.io/my-base-chart
helm repo update
helm install my-app privacera-base/privacera-base-chart

🔧 Available Charts

  • privacera-base-chart (v1.0.0) - Base microservice chart with deployment, service, ingress, and more

📋 Features

  • 🚀 Deployment and StatefulSet support
  • 🌐 Ingress with TLS configuration
  • 📊 Built-in observability (metrics, logging)
  • 🔒 Security policies and RBAC
  • 📈 Horizontal Pod Autoscaler (HPA)
  • 💾 Persistent Volume Claims
  • 🔄 Network Policies
